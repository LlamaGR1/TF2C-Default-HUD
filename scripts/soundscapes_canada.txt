//
// This is a general collection of useful soundscape utils that weren't originally designed for TF2,
// but which may fill gaps for specific settings where the existing soundscapes aren't workable.
// 
// All sounds referenced to this file are in sound/scapes, as opposed to regular TF2 soundscapes which
// are in sound/ambient. Sounds created *for* TF2 and TF2C should remain in sound/ambient.
//
// This file contains utility soundscapes (prefixed with _util) which
// aren't intended to be used directly, they can be called by other soundscapes which want
// a convenient way to add certain types of sounds. This includes soundscapes in other files,
// or in custom maps, which can inherit these and call them.
//
